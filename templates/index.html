About Transform Domain Steganography

This application provides steganography functionality using transform domain techniques:

  • DCT (Discrete Cosine Transform) - Hides data in frequency coefficients after DCT transform
  • Wavelet Transform - Embeds data in wavelet coefficients of the image or audio
  • DFT (Discrete Fourier Transform) - Hides data in frequency magnitude values
  • SVD (Singular Value Decomposition) - Embeds data by modifying singular values
  • LBP (Local Binary Pattern) - Utilizes texture patterns for steganography

These methods are more robust compared to spatial domain techniques like LSB (Least Significant Bit) substitution.

Why Transform Domain?

Transform domain steganography is more resilient to processing, compression, and filtering operations than spatial domain techniques. Instead of directly changing pixel values, these methods hide information in the transform coefficients, making the changes less detectable by statistical analysis.
